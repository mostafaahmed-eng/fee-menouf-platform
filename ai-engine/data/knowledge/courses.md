## Course Descriptions

### MATH101 - Engineering Mathematics 1
Functions, limits, continuity. Differentiation: techniques and applications. Integration: definite and indefinite integrals, techniques of integration. Applications of integration. Sequences and series. Power series, Taylor and Maclaurin series.

### MATH102 - Engineering Mathematics 2
Multivariable calculus. Partial derivatives. Multiple integrals. Vector calculus: gradient, divergence, curl. Line and surface integrals. Green's theorem, Stokes' theorem, Divergence theorem.

### CS101 - Programming Fundamentals
Introduction to programming using Python/C. Variables, data types, control structures, functions, arrays, pointers. Basic algorithms and problem-solving techniques.

### EE101 - Electric Circuits 1
Basic circuit concepts: voltage, current, power, energy. Circuit elements: resistors, capacitors, inductors. Kirchhoff's laws. Circuit analysis techniques: nodal and mesh analysis. Thevenin and Norton theorems. First-order and second-order circuits.

### EE102 - Electric Circuits 2
Sinusoidal steady-state analysis. Phasors. AC circuit analysis. Power calculations. Three-phase circuits. Frequency response. Laplace transform applications in circuit analysis. Two-port networks.

### ECE301 - Analog Communication Systems
Amplitude modulation and demodulation. Frequency and phase modulation. Noise in analog communication systems. AM and FM transmitters and receivers. Signal-to-noise ratio analysis.

### ECE302 - Digital Communication Systems
Pulse code modulation. Delta modulation. Digital modulation techniques (ASK, FSK, PSK, QAM). Information theory. Error detection and correction coding. Spread spectrum communications.

### CSE402 - Artificial Intelligence
Problem solving and search algorithms. Knowledge representation. Expert systems. Fuzzy logic. Natural language processing. Computer vision fundamentals. AI applications in engineering.

### CSE403 - Machine Learning
Supervised learning: regression, classification, decision trees, SVM. Unsupervised learning: clustering, dimensionality reduction. Ensemble methods. Model evaluation and selection. Practical ML projects.

### EPM403 - Power System Analysis
Power flow analysis. Fault analysis. Power system stability. Load frequency control. Economic dispatch. State estimation in power systems.
